## 安装

- 首先需要安装 `nodejs` ，然后通过 `npm` 安装 `gitbook-cli`

	``` dos
	npm install -g gitbook-cli
	```
	
	
## 使用

- 初始化书籍目录

	``` dos
	gitbook init
	```

- 编译书籍

	``` dos
	gitbook serve
	```
